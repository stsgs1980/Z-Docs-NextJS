## Ключевые предупреждения

### 3a-studio-mas: src/lib/prompting/ содержит ЗАГЛУШКИ
- `src/lib/prompting/evaluation/scoring.ts` — возвращает `Math.floor(Math.random() * 40) + 60` (mock)
- `src/lib/prompting/core/techniques.ts` — только 2 техники вместо 20
- **Всегда брать из `packages/prompting/`** — там реальные эвристики

### 3a-studio-mas: Self-correction симулирован
- `executeWithReview()` возвращает hardcoded "Simulated response"

### P-MAS_init ≠ предок P-MAS-architector
- P-MAS_init = **Visual layer** (иерархия, workflow UI, dashboard, WebSocket)
- P-MAS-architector = **Orchestration layer** (prompt management, skill execution, pipeline backend)
- Они **siblings**, не parent/child

### MVP-Flow-Studio-Pro ≠ Flow-Studio-Pro
- MVP — это расширенная версия (MVP-2) с 10 нодами и 26 шаблонами
- Flow-Studio-Pro — базовая версия с 5 уникальными нодами

### prompting-v0.0 извлечён из UI-Kit
- prompting-v0.0 был `src/lib/prompting/` в UI-Kit монорепо
- UI-Kit как донор 3A Studio **не нужен** — используйте prompting-v0.0 напрямую
