## Формат обновления

Каждая сессия обновляет этот файл по шаблону:

```
## Сессия [дата]

### Что делали
- [конкретно]

### Что изменилось
- [файлы]

### Статус этапов
- P0-1: IN-PROGRESS / DONE / BLOCKED
- ...

### Следующий шаг
- [что делать дальше]
```

Это файл-истина. Если здесь TODO — значит не сделано, даже если кто-то говорит "готово".
