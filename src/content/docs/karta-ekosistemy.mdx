## Общий стек: Z.ai Code Scaffold

Подавляющее большинство проектов используют один и тот же технологический стек — **Z.ai Code Scaffold**. Это не случайность: Zai-agent-toolkit писался именно для корректной работы в песочнице Z.ai, и scaffold определяет какой стек и какая структура гарантируют стабильную работу в этой среде. Проекты, следующие scaffold, работают в песочнице без проблем; исключения — те, где scaffold не применяется (npm-пакеты, Figma-экспорты, чистые утилиты).

| Слой | Технология |
|------|------------|
| Framework | Next.js 16 (App Router, Turbopack) |
| Language | TypeScript 5 (strict) |
| Styling | Tailwind CSS 4 |
| UI Library | shadcn/ui (Radix UI primitives) |
| ORM | Prisma (SQLite для локальных, PostgreSQL для продакшена) |
| State | Zustand |
| Animations | Framer Motion 12 |
| Icons | Lucide React |
| AI | z-ai-web-dev-sdk |
| Runtime | Bun |

Исключения:
- **energy-helix-3d** — React Three Fiber / Three.js (npm-пакет)
- **Token-command-center** — чистый React 18+, zero CSS
