## Что это (из README)

Каталог 213 UI-компонентов из 17 проектов в 23 категориях с живым предпросмотром, AI-описаниями, графом зависимостей, персональными заметками и тегами.

### Возможности

- **Поиск и фильтрация** — по категориям, проектам, тегам
- **Живой предпросмотр** — Sucrase для компиляции компонентов в реальном времени
- **AI-описания** — через z-ai-web-dev-sdk
- **Граф зависимостей** — визуализация связей между компонентами
- **Избранное и заметки** — персональные теги и заметки
- **Дедупликация** — обнаружение дубликатов
- **Экспорт** — JSON/CSV

### UI

- Glassmorphism дизайн
- Dark/light тема (next-themes)
- Мобильная адаптация
- Горячие клавиши
- Подсветка синтаксиса (highlight.js)
