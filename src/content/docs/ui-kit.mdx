## Что это (из README)

Foundation Component Library с Layout Engine, Theme Engine, Anti-monolith ESLint и CLI.

### Ключевые возможности
- **Layout Engine:** 51 рецепт, useLayoutAdvice hook
- **Theme Engine:** 5 пресетов
- **Anti-monolith ESLint:** max-lines, max-use-state, no-cross-layer, no-unicode-escapes
- **CLI:** инструменты для работы с @stsgs/ui
- **WCAG 2.1 AA** accessibility
- **AI rules** для 6 платформ
