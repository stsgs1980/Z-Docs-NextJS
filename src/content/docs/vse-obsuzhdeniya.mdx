# Обсуждения, мысли, заметки

> Сюда кидаем всё что обсуждали, скриншоты, тексты, графики, идеи.
> Каждый файл — отдельная тема. Формат свободный (md, png, txt).

---

## Как пользоваться

1. Создаёшь файл в этой папке: `discussions/my-topic.md`
2. Пишешь что угодно — скриншот, цитату из обсуждения, мысль, график
3. Добавляешь ссылку в SUMMARY (ниже)
