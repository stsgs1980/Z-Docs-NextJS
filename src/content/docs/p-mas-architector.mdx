## Что это (из README)

Система визуализации и управления 26 AI-агентами в 8 ролевых группах с когнитивными формулами, real-time WebSocket обновлениями и LLM-powered выполнением workflow.

### Архитектура

- **8 ролевых групп:** Strategy, Tactics, Control, Execution, Memory, Monitoring, Communication, Learning
- **5 уровней иерархии:** L0–L4
- **6 типов связей:** command, sync, twin, delegate, supervise, broadcast
- **20 когнитивных формул:** CoT, ToT, GoT, CoVe, ReAct, Reflexion, ReWOO, MoA и др. через `@stsgs/prompting`
- **3 режима layout**

## Что содержит

### Основное приложение
- Dashboard — KPI, графики, timeline
- Hierarchy — React Flow граф агентов
- Prompt Studio — компоненты оценки промптов
- Workflows — выполнение пайплайнов

### Пакеты (монорепо)
- **packages/ui/** — UI компоненты (gauge, sparkline, force-graph, IDE layout)
- **packages/cli/** — CLI инструмент (add, list, recommend, scan, theme)
- **packages/eslint-plugin/** — правила ESLint (max-lines и др.)

### API routes
- /api/agents — CRUD
- /api/tasks — CRUD
- /api/hierarchy — дерево данных
- /api/stats — агрегированные метрики
- /api/workflows/execute — выполнение пайплайнов
- /api/prompting — интерпретация промптов
